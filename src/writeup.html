Design Decision

We wanted to visually compare the weather of two or more prominent cities using existing dataset, Hence we used line graph since time-series visualization is involved and its clearer to differentiate if less than five cities are selected. We also included a bar chart to further display the distinction. In order to get better results, we should select shorter range

Technology Used

Dash is used to make the web-application, however we used Plotly for the visualization

Other Alternatives

We started with d3.js but it was difficult to interpret given the timeline, then we moved on to chart.js, which we later found to be less flexible. Hence, we chose Plotly because it came with Dash which only required the use of Python.

Development

It took around 20 hours to learn dash and plotly which was instrumental in development of the project. We divided the work into 5 categories , with each of the team member responsible for one or two divisions.

  • Data selection (Ramya)
  • Data Manipulation (Kriti)
  • Knowledge transfer (Everyone)
  • Application development (Asjad)
  • UX and UI development (Prateek)

Project Overview

Based on the requirement and our experience we agreed to use Python for the development. Since we decided to use plotly for visualization we needed to understand the development of application in Dash. Each of the team member took their time to learn and understand development of application. Later we sat together and did knowledge transfer. One of the challenge that occurred after the development of application was, inability of Dash application to get hosted on github. To overcome this issue we hosted the Dash application on hiroku app and then in turn embedded the application on a React JS based web app that can be hosted on github. The total time taken to develop this application was somewhere around 90 hours, though majority of it was utilized in learning the development in Dash and plotly. The dataset was acquired from National Centers for Environmental Information, National Oceanic and Atmospheric Administration (NOAA).
